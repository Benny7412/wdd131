Site Plan - Best Texas BBQ

Site Name

Best Texas BBQ

I chose this name because the website focuses on providing high-quality and sortable, BBQ recommendations.

Site Purpose

The purpose of this website is to have a place you can look to find the best Texas BBQ spots if you live in texas or ever plan on coming here

Scenarios

  • Where can I find the best BBQ in Texas?
  • Where can I find BBQ spots closer to me?

Text Color: #ececec; For readability (current text color)

Typography

Headings: Playfair Display

Body Text: Lato (This is what we're currently using)
